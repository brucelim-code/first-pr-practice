# first-pr-practice

A tiny sandbox repo for practicing the GitHub PR workflow: branch, commit, push, open PR, merge.

## What's here

`greet.py` — a small script with one function, `greet(name)`, and a couple of tests in `test_greet.py`.

## Usage

```bash
python greet.py Bruce
```

## Running tests

```bash
python -m unittest test_greet.py
```
